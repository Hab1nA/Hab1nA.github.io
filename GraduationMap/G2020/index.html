🏫 项目简介

天府七中 G2020 级蹭饭图是一个展示 2020 届高中毕业生就读大学地理分布的可视化工具。以天地图卫星影像为底图，将全年级四个班同学的大学去向以彩色标记呈现在全国地图上——散落各地的求学轨迹，一张图尽收眼底。无论你身处何方，地图上总有一颗属于你的坐标。

🔒 身份验证

页面加载后首先展示验证遮罩，需输入正确的学校名称才能解锁地图内容。口令通过浏览器端 SHA-256 哈希比对验证，代码中不储存任何明文口令，确保数据仅在班级内部流转。

🗺️ 班级筛选

  • 右下角「班级筛选」面板提供了 1–4 班的复选框，可单独或任意组合勾选。
  • 每个班级以独立颜色标识——1班紫色、2班绿色、3班琥珀色、4班红色。
  • 若多个班级有同学就读同一所大学，该标记会自动合并为灰色，信息窗内分班列出所有同学。
  • 面板底部实时显示当前已勾选班级的总人数。

📍 大学定位逻辑

每个大学的坐标通过以下优先级链条确定：

  1. 预置经纬度优先：若数据文件中某同学提供了有效的 latitude / longitude 字段（十进制度，纬度 -90~90、经度 -180~180），则直接以此坐标落点，无需额外网络请求。
  2. 坐标冲突检测：同一大学若不同同学提供了不一致的坐标，系统判定为冲突，自动回退到地理编码搜索。
  3. 天地图地理编码：以上情况均不满足时，通过天地图地理编码服务将"城市+大学名"转换为经纬度坐标。编码请求以 250ms 间隔逐条发送，避免触发 API 限流。

地理编码结果会缓存至内存（geoCache），同一大学只需请求一次。编码过程中页面展示加载蒙层与进度提示。

🔍 搜索功能

顶部搜索框支持两种检索模式，按优先级依次匹配：

  1. 同学姓名搜索：在全年级数据中模糊匹配姓名（不区分大小写）。优先精确匹配，命中后地图自动飞至该大学并打开信息窗。若该生所在班级未被当前勾选，会临时创建标记并自动合并显示所有班级数据。
  2. 大学/城市搜索：若姓名未命中，则按大学名称模糊匹配。同样优先精确全名匹配，支持城市名称检索。

当存在多个搜索结果时，顶部出现导航条（ ← 上一个 / 下一个 → ），可逐一浏览。关闭搜索后临时标记自动清除，原有标记恢复原状。

📊 数据统计

点击顶部「数据统计」按钮打开统计面板，内容包括：

  • 年级总览：全年级总人数及各班级人数徽标。
  • 城市分布 Top 8：以水平柱状图展示同学去向最多的城市排名。
  • 热门院校 Top 8：以水平柱状图展示录取人数最多的院校排名。院校名称智能合并——去除括号内校区后缀（如"某某大学（北京校区）"），同时保留独立办学实体的完整名称（如"中国地质大学（武汉）"与"中国地质大学（北京）"分别统计，中国石油大学、中国矿业大学同理）。

⚠️ 数据缺失

部分同学可能缺少大学去向信息。勾选班级后，右下角会显示「数据缺失」面板，点击可查看当前已勾选班级中缺失信息的具体名单。该面板独立于班级筛选，仅在有缺数据时可见。

💡 交互提示

  • 悬停标记：鼠标悬停地图标记，弹出轻量预览框，显示大学名称与就读人数。
  • 点击标记：点击标记打开完整信息窗，按班级分组列出所有同学姓名，附带班级颜色徽标。
  • 长校名滚动：标记信息窗中，大学名称超过 15 字时自动启用跑马灯滚动显示。
  • 模态框操作：所有弹窗（关于、数据统计、数据缺失）均可通过右上角 ✕ 按钮、点击背景遮罩、或按下 ESC 键关闭。
  • Toast 提示：操作反馈（搜索无结果、API 额度提示等）以底部短暂浮条形式呈现，3 秒后自动消失。
  • 比例尺：地图左下角提供比例尺控件，方便判断实际距离。
  • 缩放：支持鼠标滚轮缩放（4–18 级），初始视野覆盖中国全境。

🛡️ 容错机制

  • API 额度检测：通过监听天地图瓦片加载失败和地理编码连续失败两种通道，判定当日 API 调用额度是否已满，一旦触发即暂停队列并提示用户。
  • 超时保护：每条地理编码请求设置 8 秒超时，超时后不缓存结果，允许后续重试。
  • 版本号隔离：切换班级、关闭搜索等操作会递增版本号，使飞行中的异步回调自动失效，避免过期数据污染当前视图。

⚙️ 技术栈

  • 地图引擎：天地图 JavaScript API v4.0（卫星影像底图）
  • 地理编码：天地图 Geocoder 服务
  • 密码学：Web Crypto API（SHA-256）
  • 纯原生 HTML / CSS / JavaScript，无第三方框架依赖

🛠️ 数据维护

班级数据分别存储在 data/class1.js ~ data/class4.js 中。每个文件导出两个数组：

  • classNData：已知去向的同学，每项包含 { name, university, city } 及可选的 { latitude, longitude } 坐标字段。
  • classNMissingData：缺失去向的同学，每项包含 { name }。

若地图无法正常加载，请在 index.html 末尾将天地图 API 引用中的 TK 替换为您的有效开发密钥。
