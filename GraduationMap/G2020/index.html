关于本站

🏫 项目简介

天府七中G2020级蹭饭图是一个展示2020届高中毕业生就读大学地理分布的可视化工具。通过天地图，直观呈现同学们散落在全国各地的求学轨迹。

📖 使用说明

  • 勾选右下角班级复选框，即可在地图上显示该班同学的就读大学标记。
  • 点击地图上的彩色标记，可查看该大学就读的同学信息。
  • 鼠标悬停标记，可预览大学名称与人数。
  • 使用顶部搜索框可快速检索同学姓名，并定位到其所在大学（也支持城市或大学搜索）。
  • 可同时勾选多个班级进行对比查看。

⚙️ 配置说明

本站使用天地图 JavaScript API v4.0。若地图无法正常加载，请在 index.html 末尾将 YOUR_API_KEY 替换为您的天地图开发密钥（TK）。

申请地址：lbs.tianditu.gov.cn

🛠️ 数据维护

班级数据分别存储在 data/class1.js ~ data/class4.js 中，每个文件包含 { name, university, city } 格式的同学信息，也支持可选 { latitude, longitude } 字段；当提供经纬度时会直接按该坐标落点，不再使用搜索定位。
